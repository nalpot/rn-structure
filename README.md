# RN-Structure
react-native structure template

# Setup

```
npx react-native init <YOUR-APP-NAME> --template https://github.com/nalpot/rn-structure.git
```
```
npx react-native link
```
```
npx pod-install
```

# Run
ios
```
npx react-native run-ios
```
android
```
npx react-native run-android
```
